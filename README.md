# SetupDevMachine
SetupDevMachine - Extended Developer Workstation Installer installs various third-party components that allow you to develop websites, write code for software, host web pages and more with ease. While there initially three versioons of this, only this one will be made available, while the other two are scrapped.

It is recommended to view this before using it, as each individual's system is different from the others. **it is recommended to run this with Administrator** otherwise it won't work.

PowerShell may complain about permissions, simply use: **Set-ExecutionPolicy Bypass -Scope Process -Force** then continue.
